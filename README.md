# Hospital Outpatient Appointment Optimisation System

## 🏥 Project Overview
This project addresses a **28% DNA (Did Not Attend) rate** and a **10-week patient waiting list** at a hospital outpatient department. I designed a digital solution to automate reminders and waitlist reallocation.

## 🚀 Key Business Objectives
* **Reduce DNA Rate:** From 28% to 10%.
* **Optimise Utilisation:** Increase clinic slot usage from 65% to 85%.
* **Reduce Wait Times:** Cut the average 10-week wait down to 6 weeks.
* **Efficiency:** Achieve a 40% reduction in manual administrative workload.
